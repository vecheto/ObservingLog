Observing Log Tutorial

Welcolme! If you are observing please follow this recomendations to write a informative Log of Observations. To write a new mesagge you must write your text in the index page (/index.html) uploading an image is optional. The hour and day are saved automatically in UTC.

  1. Write a mesagge when you start your observing night. Example: "Starting observations"
  2. When the calibrations are taken, write another mesagge. Example:"Calibrations done, without problems"
  3. For every target, write a mesagge with the name of the target, filter, exposure time and every information that you consider relevant for a posterior data reducer.
  4. At least once per night, post a picture of the climate conditions.
  5. If you have technical issues or bad weather, post a picture.
  6. At the end of the night or if you have to leave the observation, post a message
  7. Remember to save the HTML as a PDF at the end of the observation or the Log will be erased
  8. To start a new log, just delete the content of "data.csv" and static directory.
  9. If you need to delete a mesagge, delete the row in "data.csv"
